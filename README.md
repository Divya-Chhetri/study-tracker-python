# Study Tracker

A Python-based study tracker using file handling and menu-driven programming.

## Features
- Add study sessions
- View study history
- Count total study sessions
- Search study subjects
- Persistent data storage using files

## Concepts Used
- Python functions
- Loops
- File handling
- Dictionaries
- Input validation

## Future Improvements
- GUI version
- Database integration
- Analytics dashboard
